DKK to AUD A simple app built with Go that provides historical data of the exchange rate for Danish Krone to Australian Dollar. The date range covers from my arrival in Denmark to assist in decisions on best time to send money back to Australia. Back end is built with Go with data displayed via HTML/CSS. A Redis container is used to cache results for up to six hours to reduce calls for data. Currently deployed on Heroku using a multi-stage Docker build. link to repo
Movie Search Component A React component (widget) to display movies, TV shows, and people using The Movie DB API. I used TDD/BDD to build a component that Fetches trending media from API when widget is first loaded. Users are able to search by name and results are filterable by movie, tv show and person. There are CSS rules for both mobile and desktop and for truncating long titles and descriptions. Also features image thumbnail placeholders and a simple caching system for storing prior search queries and results. link to repo
